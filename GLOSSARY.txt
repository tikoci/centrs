# Project glossary and cspell dictionary
#
# This file is the shared source of truth for project vocabulary, short term
# definitions, and cspell dictionary entries.
#
# `%term` in prompts or notes means "look up the matching entry or comment here."
#
# CSpell dictionary syntax supported here:
# - `~word` => use the word only when case sensitivity is off
# - `+word` or `word+` => allow optional compounds
# - `-word` or `word-` => require the word to appear in a compound
# - `!word` => explicitly forbid the word
#
# Correctly spelled words may still belong here when the attached comment is the
# canonical repo-specific definition or reference for the term.
#
